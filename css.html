CSS

É possível adicionar o estilo tipo CSS na página através de 3 modos diferentes:

  • O 1° é através do link, com o comando link no html rel=stylesheet e o href=link-do-arquivo.css. Essa forma é a mais utilizada e mais semântica. É necessário sempre colocar o link dentro da tag head.
  • O 2° modo é através da tag style dentro da tag head. É possível estilizar colocando o type=text/css e dentro da tag coloca os elementos que se quer estilizar assim como em um arquivo css.
  • O 3° modo e o menos utilizado é colocando o estilo diretamente na tag que se quer estilizar.

Alguns comandos iniciais são:

  • Text-align: que posiciona os objetos, podendo ser ao centro, ao lado
  • Display: modifica como as "caixas" da página se comportam, por ex.: se a tag é uma tag inline, ela pode mudar para uma tag block e vice-versa
