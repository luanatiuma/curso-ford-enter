Comandos do GIT

git status - verifica o arquivo e os commits

git add . - adicionar as alterações

git commit -m "" - commita as alterações adicionadas, e o que fica entre as "" é a mensagem que vai aparecer no seu commit

git push - envia todos os commits e alterações para o repositório

git pull - sincroniza as pastas e projetos que não foram adicionadas no meio ambiente, ou seja, as alterações foram realizadas em outro lugar, ex: o readme feito direto no github *o git push também sincroniza as alterações quando realizadas alterações em um local diferente do repositório, por ex.: se você realizar alteração em uma branch diferente diretamente no VSCode e quiser sincroziná-las com o repositório do github, realizando o push você sincroniza o repositório local e o repositório do github

git clone - copiar o link HTTPS do repositório desejado, dentro do github, e colocar o link ao lado do comando git clone para clonar o repositório para o local de trabalho *no terminal não é possível utilizar o atalho do techado CTRL+V, então para colar o link é necessário somente clicar com o botão direito do mouse

git branch - criar uma branch, esse comando é muito útil para realizar atualizações em um site e deixar a página online para os usuários antes da atualização de fato

git checkout - muda o branch, ex: se é realizado uma branch dev e você deseja acessá-lo, digitar o comando git checkout dev

git --version - verifica a versão do git que está instalada

git update-git-for-windows - atualiza o git para a última versão disponível

git config --global user.email "e-mail" - comando para realizar o login no git

git config --global user.name "name" - comando para realizar o login no git
